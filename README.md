## zero-ui

Open source UI library for zero projects.

### Setup LeptosFMT

```
cargo install leptosfmt
```

```
// .vscode/settings.json
"rust-analyzer.rustfmt.overrideCommand": ["leptosfmt", "--stdin", "--rustfmt"]
```
